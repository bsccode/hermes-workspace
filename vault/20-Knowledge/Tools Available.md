# Tools Available

## Core
- **terminal** - Shell commands
- **file** - read_file, write_file, patch
- **session_search** - Search past conversations
- **memory** - Persistent memory storage
- **skills** - Load/use specialized skill sets
- **todo** - Task management

## Web & Browser
- **browser** - Full browser automation (navigate, click, type, snapshot)
- **web** - Web search and content extraction

## GitHub
- **codebase-inspection** - pygount for LOC, repo analysis
- **github-auth** - Authentication setup
- **github-code-review** - Diff analysis, inline comments
- **github-issues** - Create, manage, triage issues
- **github-pr-workflow** - Branch, commit, PR lifecycle
- **github-repo-management** - Clone, fork, configure repos

## Observability
- **console** - Browser console inspection, JS errors
- **vision** - Screenshot + vision AI analysis
- **get_images** - List all page images

## Automation
- **cronjob** - Scheduled tasks with skills attached
- **delegate_task** - Spawn subagents for parallel work
