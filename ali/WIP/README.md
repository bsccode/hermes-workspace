# Node-scoped WIP

This directory is for unfinished work that should be visible across nodes without turning `main` into a pile of checkpoint commits.

Conventions:
- each node uses `ali/WIP/<node>/STATUS.md`
- the authoritative checkpoint lives on git branch `wip/<node>`
- the status file is generated by `ali/session-sync.sh end`
- once a WIP branch is merged into the default branch, `ali/session-sync.sh` clears the merged remote WIP branch automatically
